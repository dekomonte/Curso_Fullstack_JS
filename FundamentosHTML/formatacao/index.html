Elementos de Formatação

No HTML podemos escrever utilizando negrito, itálico ou ambos para destacar partes de um texto.

É importante saber que as tags de negrito e itálico existem, mas raramente são utilizadas assim. Geralmente, utiliza-se o CSS para isso.
Outras tags importantes que podem ser utilizadas no lugar de b e i: negrito e itálico.
